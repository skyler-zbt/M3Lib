# M3Lib

> M3Lib（Modern · Module · Math） 一个模块化的图形学数学库。

M3Lib的目标是成为“现代化、模块化的 GLM”，为图形学、游戏引擎与实时渲染领域提供更符合现代 C++ 理念的数学基础设施。

## 项目目标

M3Lib 希望成为：

- 一个完全基于 Modules 架构的现代 C++ 数学库
- 更契合现代 C++26 设计思想的 GLM 替代方案

与传统数学库相比，M3Lib 更加注重：

- 完整的 C++ Modules 支持
- GLSL 风格的接口与行为规范
- 清晰且一致的 API 设计
- 强类型安全
- 合理的内存对齐设计
- 对现代 SIMD 及编译器优化能力的利用
- 贴近 GPU / Shader 风格的数学接口

在可行范围内，M3Lib 将尽可能遵循 GLSL 规范与行为，以减少 CPU 端与 GPU 端数学代码之间的差异，为开发者提供更一致的编码体验。

---

## 计划特性

- 向量类型（`vec2`、`vec3`、`vec4`）
- 矩阵类型（`mat3`、`mat4` 等）
- 四元数支持
- GLSL 风格的数学函数
- 基于 C++ 标准 SIMD（`std::simd`）的支持
- 内存对齐优化
- 对 `constexpr` 友好的设计
- 纯模块架构
- 跨平台支持
- 基于 xmake 的现代化构建系统

---

## 环境要求

- 支持 C++26 的编译器
- 完整的 C++ Modules 支持
